# [u11g004]商品棚卸開始

## 目的

- 棚卸表の開始・中止の状況制御を行う。

#### 商品棚卸開始

- 棚卸中は商品入荷、売上登録など商品移動を伴う登録は不可となる。
- 商品棚卸登録画面で商品棚卸登録ができるようになる。

#### 商品棚卸中止

- 棚卸中は商品入荷、売上登録など商品移動を伴う登録が可能となる。
- 中止した場合、登録済みの棚卸結果は削除されず、商品棚卸を再度開始すると続きから作業可能。

## ユースケース

### 商品棚卸開始

- ユーザー:メニューの商品棚卸リンクを押下
- システム:商品棚卸開始画面を表示
- ユーザー:部署コードを入力
- ユーザー:部署選択ボタンを押下
- ユーザー:棚卸開始ボタンを押下
- システム:部署の棚卸ステータスを棚卸中に変更

### 商品棚卸中止

- ユーザー:メニューの商品棚卸リンクを押下
- システム:商品棚卸開始画面を表示
- ユーザー:部署コードを入力
- ユーザー:部署選択ボタンを押下
- ユーザー:棚卸中止ボタンを押下
- システム:部署の棚卸ステータスを通常に変更
